create_config(number_of_trees, data_size_for_tree)¶

Configures forest

Parameters

  • number_of_trees (int) – Number of trees for the forest to contain.

  • data_size_for_tree (int) – Determines number of rows of training data that go into every tree construction.

class random_forest.forest.Forest(data)¶

bootstrap(data)¶

Randomly draws with repetitions rows form given data. Record are stored in list of np.ndarrays form which trees are created. Record that were not selected can by found in self.out_of_bag.

Parameters

data (pandas.DataFrame) – training data for the forest

Returns

list of np.ndarrays for each tree training

Return type

list
